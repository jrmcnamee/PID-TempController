# PID-TempController
Lab project to control the temperature of water using an Arduino and a PID algorithm

## Proportional-Integral-Derivative Algorithms
A Proportional-Integral-Derivative Algorithm can be used in controllers to optimize computer signals when controlling some system. A PID is a classic closed-loop feedback system, meaning it takes an input and then systematically compares current state to its input over time.
